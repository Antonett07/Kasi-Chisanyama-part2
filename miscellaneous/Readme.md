Kasi Chisanyama Website
This is a static HTML website for the chisanyama, a local fast-food resturant in Limpopo, Burgersfort, South Africa.

## Website Structure
- Homepage (index.html): Welcome section, specialties overview, and call to action
- About Us (about.html): Company story, mission, values, and team information
- Menu (menu.html): Detailed menu with braaied meat, cooked pap, traditional dishes and beverages
- Contact (contact.html): Location details, contact information, and interactive map
- Enquiries (enquiry.html): Contact form for customer inquiries and questions
- Services (services.html): The kind od services we offer, dining experiences, takeaways, deliveries, events and catering services

## Features
Responsive navigation across all pages
Visual elements showcasing the dishes we serve and dining experiences
Contact form with multiple enquiry types
Google Maps integration for location finding
Consistent branding and footer across all pages

## Technical Details
Built with HTML5
Semantic markup for better accessibility
Responsive design considerations
Relative paths for asset management
